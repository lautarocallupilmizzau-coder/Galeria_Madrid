🌆 Galería de Madrid
Una pequeña galería interactiva creada como proyecto académico, utilizando HTML, CSS y JavaScript. El objetivo fue recrear una composición visual similar a un ejemplo dado, mostrando una imagen principal acompañada por cuatro miniaturas que permiten cambiar dinámicamente la fotografía destacada.

✨ Características del proyecto
Imagen principal de 613×460 px

Cuatro miniaturas de 133×100 px, alineadas verticalmente

Separación exacta de 20 px entre miniaturas

Cambio dinámico de la imagen principal al hacer clic en una miniatura

Diseño simple, limpio y fiel a la consigna

Código organizado en un único archivo HTML con estilos y scripts integrados

🖼️ Tecnologías utilizadas
HTML5 para la estructura

CSS3 para el diseño y la disposición

JavaScript para la interacción

Imágenes reales de Madrid proporcionadas en la carpeta de trabajo

🎯 Objetivo académico
Este proyecto fue desarrollado como parte de una actividad práctica para aprender a:

Manipular imágenes en una página web

Controlar tamaños y alineaciones con CSS

Implementar interactividad básica con JavaScript

Organizar un proyecto web y prepararlo para entrega
